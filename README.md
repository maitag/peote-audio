# Peote Audio - haxe lime library to wrap around OpenAL and WebAudio

This library is to simplify usage of OpenAL and WebAudio and is still work in progress.
  
It's written in [Haxe](http://haxe.org) and runs multiplatform with [Lime](https://github.com/openfl/lime)s [OpenAL](https://github.com/openfl/lime/tree/develop/src/lime/media/openal) and environment.  




  
## Installation
```
haxelib git peote-audio https://github.com/maitag/peote-audio.git
```
